🔔 LINE 到價通知

📖 首次設定教學（約 10 分鐘）
  1. 到 LINE Developers Console 用你的 LINE 帳號登入
  2. 建立 Provider → 建立 Messaging API channel（名稱隨意，例如「台股到價通知」）
  3. 在 channel 的 Messaging API 分頁最下方，發行 Channel access token（交給管理者設定到伺服器）
  4. 用手機 LINE 掃描該分頁的 QR code，把官方帳號加為好友
  5. Webhook URL 設為 https://my-stocks-api.david870114.workers.dev/line-webhook 並開啟「Use webhook」
  6. 對官方帳號傳任意訊息，它會回覆你的 LINE User ID，貼到上面欄位並儲存即完成
